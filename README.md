# nathantranquilla.me

Personal blog and website for Nathan Tranquilla.

## Quickstart

### Using Flox (Recommended)

```sh
# Install flox (if not already installed)
curl -L https://install.flox.dev | sh

# Activate the development environment
flox activate

# Start development server
pnpm dev
```

## Rake Tasks

```sh
# Show available tasks
rake
```
